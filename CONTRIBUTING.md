# Contributing

Thanks for your interest in contributing!

## How to contribute

1. **Fork** the repository  
2. Create a new branch:
   ```
   git checkout -b feature/my-feature
   ```
3. Make your changes  
4. Commit your work:
   ```
   git commit -m "feat: describe your change"
   ```
5. Push your branch:
   ```
   git push origin feature/my-feature
   ```
6. Open a **Pull Request** describing the improvements

---

## Code Style Guidelines
- Use clear, readable Java code  
- Keep classes and methods small and focused  
- Use meaningful variable and method names  
- Write comments only for complex logic  

---

## Testing
Add unit/integration tests when you add new functionality or fix bugs.

---

## Questions?
Feel free to open an issue or reach out!
